# HIMATIK DSS Documentation

This folder contains current project documentation for the Laravel recruitment and SPK system.

- `project-structure.md`: codebase map and verification commands.
- `blade.md`: Blade/UI route and view documentation.
- `api.md`: API endpoint documentation based on `routes/api.php` and API controllers.

Primary in-app docs:

- Blade docs: `/docs/blade`
- Scribe API docs: `/docs`
- L5 Swagger UI: `/api/documentation`
